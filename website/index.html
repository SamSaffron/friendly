Friendly

Store schemaless data in MySQL.

Add fields and indexes without blocking tables and cache objects without breaking a sweat.

  • Store schemaless data in MySQL.
  • Build indexes offline without affecting production.
  • Add and remove fields on big datasets without long running migrations.
  • Write-through and read-through caching baked right in (it's fast).
